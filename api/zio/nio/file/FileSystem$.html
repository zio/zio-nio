  6.  def default: FileSystem

    The default filesystem.

    The default filesystem.

    The default file system creates objects that provide access to the file systems accessible to the Java virtual machine. The working directory of the file system is the current user directory, named by the system property user.dir.

    Note: The default filesystem cannot be closed, and if its close is called it will die with UnsupportedOperationException. Therefore, do not use resource management with the default filesystem.
